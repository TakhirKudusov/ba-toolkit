## How it works

Most skills follow the same cycle: **Command → Context → Interview → Generate → Refine**. Each skill loads all previous artifacts plus the domain reference, asks targeted questions (one at a time, table of options, one marked **Recommended**), writes a Markdown artifact, and offers refinement subcommands before moving on.

Every artifact links back to its predecessors, forming the chain `FR → US → UC → AC → NFR → Entity → ADR → API → WF → Scenario → Task`. Run `/trace` to verify coverage and `/analyze` for severity-rated findings.

After `/handoff`, run `/implement-plan` to produce a **phase-and-DAG implementation plan** an AI coding agent can execute step by step — every task references the FR / US / AC it implements and carries its own Definition of Done.

## Quick install

```bash
npx @kudusov.takhir/ba-toolkit init
```

Prompts for project name, domain, and AI agent, then creates `output/{slug}/`, `AGENTS.md`, and installs the 24 skills into your agent's directory.
