## How it works

Most skills follow the same cycle: **Command ‚Üí Context ‚Üí Interview ‚Üí Generate ‚Üí Refine**. Each skill loads all previous artifacts plus the domain reference, asks targeted questions (one at a time, table of options, one marked **Recommended**), writes a Markdown artifact, and offers refinement subcommands before moving on.

Every artifact links back to its predecessors, forming the chain `FR ‚Üí US ‚Üí UC ‚Üí AC ‚Üí NFR ‚Üí Entity ‚Üí ADR ‚Üí API ‚Üí WF ‚Üí Scenario ‚Üí Task`. Run `/trace` to verify coverage and `/analyze` for severity-rated findings.

After `/handoff`, run `/implement-plan` to produce a **phase-and-DAG implementation plan** an AI coding agent can execute step by step ‚Äî every task references the FR / US / AC it implements and carries its own Definition of Done.

## Quick install

```bash
npx @kudusov.takhir/ba-toolkit init
```

Prompts for project name, domain, and AI agent, then creates `output/{slug}/`, `AGENTS.md`, and installs the 24 skills into your agent's directory.
